# How to run
- Install extension in your browser [Violentmonkey](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag?hl=be)
- In order for **Rocky Rabbit** to open in a browser, install [this script](https://github.com/mudachyo/Rocky-Rabbit/raw/main/rocky-web.user.js)
- To install **autoclicker** install [this script](https://github.com/mudachyo/Rocky-Rabbit/raw/main/rocky-autoclicker.user.js)
- Open [Bot Rocky Rabbit](https://web.telegram.org/k/#?tgaddr=tg%3A%2F%2Fresolve%3Fdomain%3Drocky_rabbit_bot%26appname%3Dplay%26startapp%3DfrId2475526) and run the game
